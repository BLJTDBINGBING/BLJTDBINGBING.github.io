Lammps及Ovito学习记录及问题处理

  本人目前就读于北京航空航天大学机械工程及自动化学院的机器人工程专业出于内卷要求及在和蔼可亲以及可爱的燕鑫老师的指导下，我选择在大三从事分子动力学的研究。

  由于分子动力学在对实验的技术要求以及设备要求极高，因此我们多采用数值仿真的形式去对分子动力学进行研究，以下内容主要记录我的仿真软件学习记录和问题处理。

PS：本人刚刚入门，如有表述不准或大错特错，欢迎指正，我一定虚心求教！

Linux篇

目录切换命令

pwd —— #显示当前所在目录

cd  ~ —— #切换到主目录

cd  dir —— #切换到指定目录

cd .. ——#切换到上一级目录

文件展示命令

ls —— #展示当前目录下所有的文件和文件夹

ls -a—— #展示当前目录下所有文件和文件夹（含隐藏文件）

创建和查看文件的命令

touch file —— #创建文件file

mkdir dir —— #创建目录dir

cat file —— #查看文件file内容

more file —— #查看文件file内容

head file —— #查看文件file前十行

tail file —— #查看文件file后十行

删除命令和重命名或移动命令

rm file —— #删除文件file

rm -r dir —— #删除目录dir

mv file1 file2—— #若file2不存在，则将file1改名为file2；
#若file2存在，则将file1覆盖掉file2，并更名为file2

mv file dir —— #将文件file移动到dir目录中

Vi编辑器（linux系统的标准编辑器）

分为三个模式，在一个模式下只能使用该模式具有的功能

1） 插入模式（Insert mode）：此模式下可以输入字符。按ESC将回到命令模式

2） 底行模式（Last line mode）：可以保存文件、退出vi、设置vi、查找等功能

3） 命令模式（Command mode）：可以移动光标、删除字符等

终端窗口使用vi

vi file ——#使用vi编辑文件file
若文件不存在，则创建文件file

常用的vi编辑器命令

在命令模式下使用

：w —— #保存文件

：q —— #退出编辑器，如果文件已修改请使用下面的命令

：q！ —— #退出编辑器，且不保存

：wq —— #推出编辑器，且保存文件

a —— #在当前光标位置的右边添加文本

i —— #在当前光标位置的左边添加文本

o —— #在当前行的下面新建一行

在插入模式下使用

esc —— #切换到命令模式

Nano编辑器

终端窗口使用nano

nano file —— #使用nano编辑文件file，若文件不存在则创建文件file

nano编辑器常用快捷键

ctrl+o —— #保存当前文档（ctrl+s也可以）

esc+u —— #撤销上次操作

ctrl+u —— #粘贴

ctrl+g —— #打开nano帮助文档

快速查找

grep —— #查找固定形式的行

解压

tar -zxvf file —— #压缩或解压缩
